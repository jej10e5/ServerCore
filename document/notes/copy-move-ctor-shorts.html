이거 왜 필요해?
nullptr 없으면 일어나는 일
1. rhs (임시값) 소멸자 호출
2. Release() → refCount-- → 0
3. delete this 실행!
4. b는 이미 그 주소를 들고 있음
5. → Use-After-Free 💀
nullptr이면?
if (_ptr != nullptr) 체크에서 걸려
Release()가 아무것도 안 함 ✓
자동 선택
컴파일러가 알아서
오버로드를 고른다
// lvalue → 복사 생성자
TSharedPtr<Player> b = a;

// rvalue → 이동 생성자
TSharedPtr<Player> b = std::move(a);

// 함수 반환도 rvalue → 이동
TSharedPtr<Player> b = GetPlayer();
두 시그니처만 잘 만들면
컴파일러가 인자 타입(lvalue/rvalue)에 따라
자동으로 맞는 오버로드를 선택한다
